# snowflakeauth: Authentication Helpers for 'Snowflake'

Authentication helpers for 'Snowflake'. It provides compatibility with
authentication approaches supported by the 'Snowflake Connector for
Python' <https://pypi.org/project/snowflake-connector-python> and the
'Snowflake CLI' <https://pypi.org/project/snowflake-cli>.
